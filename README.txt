- Cách để khởi động phần mềm.
- Để chạy được chương trình, máy tính cần có Python để chạy. Có thể sử dụng lệnh trong command line " python --version ".
Nếu máy đã có python thì chương trình sẽ trả về phiên bản Python đang chạy.

- Kiểm tra pip " pip -V". Phiên bản của pip sẽ được hiển thị nếu nó đã được cài đặt thành công. Có thể tham khảo cách cài đặt
từ các trang web. Thông thường, pip sẽ có sẵn chung với phiên bản Python.

- Cài đặt thư viện Tkinter bằng lệnh " pip install tk ". Lệnh này sẽ bắt đầu tải xuống và cài đặt các gói liên quan đến thư viện Tkinter.

- Sau khi có đầy đủ các phần cần thiết. Có nhiều cách để chạy chương trình.
	+ Thứ nhất, nhấp đúp vào file app.py.
	+ Thứ hai, mở command line , điều hướng đến thư mục và gõ ,lệnh python app.py.
	+ Thứ ba, sử dụng Visual Studio Code insert thư mục vào, và chạy file app.py.
- Có thể insert các bảng câu đố từ bên ngoài, hoặc ấn nút play để chơi. Các bảng câu đố có sẵn được lưu trong file gamemode.py
- file engine_solver.py dùng để lưu các giải thuật dùng chính để giải câu đố.
